# Story 2.9 — Editor de Código (Monaco)

## Objetivo

Integrar o Monaco Editor como playground de codigo no Web App. Escrever, editar e executar JavaScript no browser com syntax highlighting, tema Matrix, output visivel e execucao em Web Worker sandbox.

## Acceptance Criteria

- [x] AC1: Pagina `/playground` com layout de 2 paineis (editor + output), resizavel
- [x] AC2: Monaco Editor com tema Matrix customizado (fundo escuro, texto verde, keywords ciano)
- [x] AC3: Editor configurado para JavaScript com syntax highlighting, bracket matching, auto-indent
- [x] AC4: Botao "Executar" executa codigo em Web Worker sandbox com timeout de 5s
- [x] AC5: Painel de output: console.log verde, erros vermelho, warnings dourado, timestamps
- [x] AC6: Erros de sintaxe com squiggly underline no editor + output
- [x] AC7: Botoes Resetar, Copiar Codigo, Tela Cheia
- [x] AC8: Exercicios do chat abrem playground pre-carregado via query param

## Dependencies

- Story 2.5 (Chat Interface) ✅
- Story 2.6 (Lesson Flow) ✅
