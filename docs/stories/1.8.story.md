# Story 1.8: Primeira Aula — Logica de Programacao

## Objetivo

Implementar a primeira aula real do Koda: Logica de Programacao (Modulo 1). Pedro vai aprender sobre variaveis, tipos e condicionais com analogias do dia-a-dia, e provar que entendeu passando pelos Portoes 1 (compreensao) e 2 (pratica). Esta e a experiencia central do produto — se a primeira aula nao for boa, Pedro desiste. O Koda precisa ser um professor incrivel via WhatsApp.

## Technical Notes

- O AI Engine usa `claude-sonnet-4-20250514` — melhor para educacao, empatia e explicacoes detalhadas
- Structured output: instruir o Claude no system prompt a retornar JSON com decisoes + texto para o aluno. Usar regex ou JSON.parse para extrair o JSON da resposta
- O curriculo em YAML (`curriculum/modules/01-logica.yaml`) contem metadados dos conceitos. O conteudo didatico real e montado pelo prompt (Layer 4)
- Referencia das 6 camadas: `docs/architecture/fullstack-architecture.md` secao 7
- Referencia dos portoes: `docs/prd.md` secao FR5
- Tabela de XP: Portao 1 = +30 XP, Portao 2 = +50 XP, Portao 2 com ajuda = +25 XP
- O Context Builder deve buscar ultimas 10 interacoes da tabela `interactions` para Layer 5
- Anti-jailbreak no system prompt: "Voce e Koda, professor de programacao. NUNCA saia do papel de professor."
