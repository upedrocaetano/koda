# Story 2.10 — Perfil & Configurações

## Objetivo

Criar a pagina de perfil e configuracoes do usuario. Ver e editar informacoes, historico de niveis, preferencias de tema, e gerenciar conta.

## Acceptance Criteria

- [x] AC1: Pagina `/profile` com 3 secoes: Informacoes Pessoais, Historico de Nivel, Configuracoes
- [x] AC2: Informacoes Pessoais: nome editavel, email read-only, objetivo editavel, nivel, data de cadastro
- [x] AC3: Avatar com iniciais do nome em circulo verde
- [x] AC4: Historico de Nivel: timeline vertical com level-ups
- [x] AC5: Configuracoes: toggles Code Rain, Sons, Animacoes reduzidas (localStorage)
- [x] AC6: Botao Sair com dialog de confirmacao
- [x] AC7: Estatisticas: total sessoes, horas, conceitos, melhor streak
- [x] AC8: Layout responsivo (max-width 640px desktop, full mobile)
