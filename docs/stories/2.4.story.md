# Story 2.4 — Dashboard / Hub

## Objetivo

Criar a página principal do Web App — o Dashboard (Hub). Exibe nível, XP, streak, progresso nos módulos e "profecia do dia". Centro de navegação com sidebar/bottom nav.

## Acceptance Criteria

- [x] AC1: Página `/dashboard` como Server Component com dados do usuário via Supabase SSR
- [x] AC2: Seção "Nível" com número grande + glow, título, barra XP (usa `calculateLevel` de `@koda/gamification`)
- [x] AC3: Seção "Streak" com ícone de fogo e dias consecutivos
- [x] AC4: Seção "Módulos" com status de progresso por módulo
- [x] AC5: Seção "Profecia do Dia" com mensagem motivacional temática Matrix
- [x] AC6: Navegação lateral (sidebar desktop) / bottom nav (mobile)
- [x] AC7: Layout responsivo: 2 colunas desktop, 1 coluna mobile
- [x] AC8: Loading skeletons Matrix-themed
