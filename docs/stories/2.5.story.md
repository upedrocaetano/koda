# Story 2.5 — Interface de Chat

## Objetivo

Criar a interface de chat para conversar com o Koda via web. Mensagens em balões, input de texto, histórico, indicador de digitação, e streaming de respostas.

## Acceptance Criteria

- [x] AC1: Página `/chat` com layout full-height: área de mensagens (scroll) + input fixo inferior
- [x] AC2: Componente `ChatMessage` para mensagens do usuário (direita) e Koda (esquerda) com markdown
- [x] AC3: Blocos de código com syntax highlighting e botão copiar
- [x] AC4: `ChatInput` com textarea auto-expansível, Enter para enviar, Shift+Enter para quebra
- [x] AC5: Indicador "Koda está pensando..." com animação
- [x] AC6: API Route `/api/chat` que recebe mensagem e retorna resposta
- [x] AC7: Histórico de mensagens carregado do Supabase (últimas 50)
- [x] AC8: Zustand store para estado do chat: messages[], isLoading, sendMessage(), loadHistory()

## Dependencies

- Story 2.2 ✅, Story 2.3 ✅
